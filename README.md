# CurlGet

## Install

composer project akeb/framework

Composer config

```json
{
    "require": {
        "akeb/framework": "^1.0.0"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/AKEB/Framework-AKEB-framework"
        }
    ]
}
```

or

```bash
composer require akeb/framework
```
